◆ seHayaEjecutado()

void AccionVerificable::seHayaEjecutado ( )

Corrobora si la acción dada se cumple dentro de un periodo dado en forma de sondeo con intervalos (por defaul de 1ms + el tiempo que tarde la acción en ejecutarse)

Corrobora si el retorno de la función a verificar es verdadero durante un tiempo dado por la AccionVerificable::tiempoLimite. La verificación se realiza en forma de sondeo, por defecto el tiempo entre cada verificación es de 1ms, la cantidad de verificaciones va a estar condicionado al tiempo que tarde la acción en ejecutarse. Ej: Si la duración se configura en 200ms, el intervalo en 1ms y la acción demora 19ms en ejecutarse, la verificación se realizará un máximo de 10 veces.

Para obtener el tiempo se utiliza Chil::milisegundos() por ende debe estar inicializado para poder usar este método.

Si la acción no se corrobora se lanza AccionNoEjecutadaExcepcion
